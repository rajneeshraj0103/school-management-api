# School Management API

A production-style ASP.NET Core Web API project built for learning backend development concepts step-by-step.

## 🚀 Features Implemented

- Layered Architecture
- Repository Pattern
- DTOs (Data Transfer Objects)
- AutoMapper
- JWT Authentication
- Role-Based Authorization
- Password Hashing using BCrypt
- Pagination
- Filtering
- Sorting
- Pagination Metadata
- Serilog Logging
- Global Exception Handling Middleware
- Standardized API Responses

---
